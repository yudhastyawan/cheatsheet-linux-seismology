Download Program:

#####  http://www.eas.slu.edu/eqc/eqc_cps/Download/NP330.Jan-05-2018.tgz  ########


1) First Install needed packages:
	sudo apt-get install make
	sudo apt-get install gcc
	sudo apt-get install gfortran
	sudo apt-get install xorg-dev
	sudo apt-get install libncurses5-dev
	sudo apt-get install gawk
	sudo apt-get install gv
	sudo apt-get install ImageMagick

2) uncompress "NP330.May-08-2016.zip" in your home folder

3) change dictory:
cd ~/PROGRAMS.330

4) use the following command:
	./Setup
* Then the result should be like this:

	Directory lib exists

5) For 64bit linux use:
	./Setup LINUX64
   For 32bit linux use:
	./Setup LINUX40

6) If everything is correct, enter the command:
	./C

to “make” everything.
